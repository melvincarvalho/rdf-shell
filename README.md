# rdf-shell
rdf shell

# Installation

```
    sudo npm install -g rdf-shell
```

# Usage

```
    commands
      rdf cat <uri>                    - displays a uri
      rdf help                         - shows help message
      rdf ls <uri>                     - shows files in a container
      rdf put <uri> <data>             - puts turtle data to a file    
```
